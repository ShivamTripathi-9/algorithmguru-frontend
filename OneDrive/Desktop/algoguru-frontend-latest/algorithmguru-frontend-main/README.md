# AlgorithmGuru Frontend

A production-ready React + Vite frontend for an AI project-based learning platform. Users can browse AI projects, view project details, complete step-by-step guided lessons in a code editor workspace, and track their progress through a dashboard.

---

## Tech Stack

- **React 19** with functional components and hooks
- **Vite** for fast development and optimised builds
- **React Router v7** for client-side routing
- **Tailwind CSS v4** for utility-first styling
- **Zustand** for global state (auth, editor, project)
- **@monaco-editor/react** for the in-browser Python code editor
- **react-hot-toast** for toast notifications
- **lucide-react** + **react-icons** for icons

---

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
